# Hygge - 個人生活管理平台

一個整合待辦清單、番茄鐘、天氣、筆記等功能的生活管理工具。

## 專案目標

- 學習前端開發技術（HTML/CSS/JavaScript）
- 建立個人作品集專案
- 練習 Git 版本控制

## 技術棧

- HTML5
- CSS3 (Flexbox/Grid)
- JavaScript (ES6+)
- Bootstrap 5
- LocalStorage
- Weather API
