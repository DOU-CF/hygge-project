# Hygge - 個人生活與工作管理平台

> 一個整合待辦清單、番茄鐘、天氣、筆記等功能的生活管理工具

![專案狀態](https://img.shields.io/badge/status-in%20development-yellow)
![進度](https://img.shields.io/badge/progress-10%25-blue)
![授權](https://img.shields.io/badge/license-MIT-green)

## 🎯 專案簡介

Hygge（發音：/ˈhjuːɡə/）源自丹麥語，意為「舒適、愜意的生活方式」。

本專案旨在建立一個個人生活與工作管理平台，提供：

- 📝 待辦清單管理
- 🍅 番茄鐘專注計時
- 🌤️ 即時天氣資訊
- 💧 健康提醒功能
- 📒 雲端筆記系統

### 專案目標

- 學習前端開發核心技術（HTML/CSS/JavaScript）
- 建立完整的作品集專案
- 練習 Git 版本控制與自動化部署
- **12/15 前完成 50% 核心功能**

---

## 🛠️ 技術棧

### 前端技術

- **HTML5** - 語意化標籤
- **CSS3** - Flexbox / Grid / 動畫效果
- **JavaScript (ES6+)** - 原生 JS，無框架依賴
- **Bootstrap 5** - 響應式佈局與 UI 元件

### API 與儲存

- **LocalStorage** - 本地資料持久化
- **OpenWeatherMap API** - 天氣資料串接
- **Fetch API** - 非同步資料請求

### 開發工具

- **Git** - 版本控制
- **GitHub Actions** - CI/CD 自動部署
- **VS Code** - 開發環境
- **Live Server** - 本地即時預覽

code .
